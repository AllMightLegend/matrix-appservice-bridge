getByJungleData(dataQuery) → {Promise.<Array.<JungleUser>, Error>}

Get jungle users by some data about them, previously stored via the set method on the Jungle user.
Parameters:
Name Type Description
dataQuery Object The keys and matching values the jungle users share. This should use dot notation for nested types. For example: { "topLevel.midLevel.leaf": 42, "otherTopLevel": "foo" }
Source:
  • components/user-bridge-store.js, line 188
Throws:
If dataQuery isn't an object.
Returns:
Resolves to a possibly empty list of JungleUsers. Rejects with an error if there was a problem querying the store.
Type
Promise.<Array.<JungleUser>, Error>
Example
jungleUser.set({
  toplevel: "foo",
  nested: {
    bar: {
      baz: 43
    }
  }
});
store.setJungleUser(jungleUser).then(function() {
  store.getByJungleData({
    "toplevel": "foo",
    "nested.bar.baz": 43
  })
});
