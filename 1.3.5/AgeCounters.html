Class: AgeCounters

AgeCounters

new AgeCounters()

A small helper object that counts into buckets the number of calls to its bump that fall into three age categories; ages within an hour, within a day or within seven days. Counts are maintained within the object, and can be fetched to set into a gauge metric object. This class is useful when exporting metrics that count the number of hourly/daily/weekly active instances of various types of object within the bridge.
